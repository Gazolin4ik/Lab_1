"Участники проекта из группы АБ(с) - 322: Зинин Вячеслав; Рабжуев Валерий; Черняков Илья. Описание проекта: наша команда используя Git и github создала онлайн репозиторий, в котором каждый из нас создал отдельную ветку для выполнения поставленных задач, используя язык программирования Java. Как было написано ранее наш репозиторий содержит 4 ветки: основную, в которой находятся все программы с других веток, а также программа, в которой выполняется одна из 3-х программ с других веток, и три ветки с этими программами. Сами программы представляют собой нахождение параметров 3-х геометрических фигур: прямоугольника, треугольника, трапеции."
